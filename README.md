# Hisaab-App
💰 Hisaab — Personal Expense Tracker A beautiful, offline-first finance manager built with Flutter &amp; Hive. Track daily income and expenses with custom categories, time-stamped transactions, and a sleek glassmorphism UI — completely private, no internet required.  🌍 Multi-Currency — PKR, USD, EUR, GBP, AED, SAR, INR
